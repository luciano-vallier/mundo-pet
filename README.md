# 🐾 Mundo Pet - Plataforma Online

Este projeto consiste no desenvolvimento do front-end de um sistema web para o pet shop **Mundo Pet**. O site foi desenvolvido como parte da Fase 1 do Projeto da disciplina de Fundamentos de Sistemas Web, focando na estruturação e organização de conteúdo utilizando apenas HTML puro.

## 🚀 Objetivo do Projeto
O objetivo desta fase é criar a fundação estrutural do site, apresentando o catálogo de produtos e a lista de serviços oferecidos, garantindo a navegabilidade e a clareza das informações para o usuário final.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação de todo o conteúdo.
* **Tabelas HTML:** Utilizadas para a organização e layout do catálogo de produtos e serviços.
* **GitHub Pages:** Hospedagem pública da aplicação.

## 📋 Funcionalidades da Página
A página principal (`index.html`) apresenta:
1. **Cabeçalho:** Identificação da loja e menu de navegação para as seções do sistema.
2. **Catálogo de Produtos:** Dividido em 3 categorias principais:
   * **Acessórios:** Itens de conforto e passeio.
   * **Rações:** Nutrição para cães e gatos.
   * **Higiene e Limpeza:** Produtos para cuidado animal e ambiente.
3. **Seção de Serviços:** Descrição de serviços como Banho e Tosa, incluindo valores e especificações sobre tele-busca.
4. **Rodapé:** Informações de contato, endereço, CNPJ e créditos de desenvolvimento.
