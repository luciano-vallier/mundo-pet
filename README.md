# 🐾 Mundo Pet - Plataforma Online

Este projeto consiste em um sistema web para um Petshop, desenvolvido como parte das atividades da disciplina de Desenvolvimento Web. A plataforma permite a visualização de produtos, serviços e o agendamento de banho e tosa.

## 🚀 Funcionalidades

### Fase 1: Estrutura Básica
* **Página Inicial:** Apresentação da marca e destaques.
* **Produtos:** Exibição de 3 categorias (Acessórios, Rações e Higiene) com fotos, descrições e valores.
* **Serviços:** Detalhamento dos serviços de banho e tosa, com opções de tele-busca.
* **Rodapé:** Informações de contato, endereço e autoria.

### Fase 2: Interatividade e Dinamismo
* **Estilização:** Uso de CSS3 e framework **Bootstrap 5**.
* **Componentes:** Carrossel de imagens na página inicial para promoções.
* **Formulário de Cadastro:** Cadastro completo de cliente (CPF, Endereço, etc.) e do Pet (Nome, Raça, Idade).
* **Agendamento Inteligente:** Sistema de agendamento com calendário.
* **Validação JavaScript:** Validação de horário comercial (agendamentos apenas entre 08:00 e 16:00).
* **Acessibilidade:** Uso de atributos `alt` em todas as imagens para leitores de tela.

## 🛠️ Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **Bootstrap 5** (Layout e Carrossel)
* **JavaScript** (Validação de formulários e lógica de horários)
